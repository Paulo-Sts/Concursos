# Anotações – Introdução à Inteligência Artificial

## 1. Conceituação Geral

- **Inteligência Artificial (IA):** campo multidisciplinar da Ciência da Computação.
- Objetivo: desenvolver tecnologias autônomas para executar funções que normalmente exigem **inteligência humana**, tais como:
  - Criatividade.
  - Auto aperfeiçoamento.
  - Uso da linguagem.
  - Visão computacional (reconhecimento de padrões em imagens).

### 1.1 Capacidades dos programas de IA

| CAPACIDADE | DESCRIÇÃO |
| --- | --- |
| Aprender | Adquirir conhecimento a partir de dados. |
| Raciocinar | Aplicar lógica para chegar a conclusões. |
| Reconhecer padrões | Identificar regularidades em dados. |
| Inferir | Deduzir informações a partir de evidências. |
| Tomar decisões | Escolher a melhor ação com base em dados. |
| Solucionar problemas | Encontrar respostas para questões complexas. |
| Processar grandes volumes de dados | Analisar massas de dados de forma rápida e eficiente. |

## 2. Abordagens Principais da IA

| ABORDAGEM | DESCRIÇÃO | EXEMPLO |
| --- | --- | --- |
| **IA Simbólica** | Representação de conhecimento por meio de símbolos lógicos. Conhecimento programado diretamente por humanos. | Sistemas especialistas baseados em lógica de primeira ordem (Prolog). |
| **IA Conexionista** | Baseada em redes neurais artificiais. Utiliza **Machine Learning**. Requer grandes massas de dados. | Redes neurais, deep learning. |

## 3. Tipos de IA segundo a Capacidade

| TIPO | DESCRIÇÃO |
| --- | --- |
| **IA Limitada (Estreita)** | Capacidade de desempenhar uma tarefa específica para a qual foi designada. |
| **IA Geral** | Capacidade de aprender e desempenhar **qualquer tarefa** que os seres humanos realizam. |
| **Superinteligência Artificial** | Simulação completa das funcionalidades do cérebro humano. Modelo mais avançado (ainda teórico). |

## 4. Tipos de IA segundo a Funcionalidade

| TIPO | DESCRIÇÃO | ESTÁGIO |
| --- | --- | --- |
| **Máquina Reativa** | Funciona com base em regras pré-programadas. Não aprende com experiência nem constrói memória. | Forma mais antiga e limitada. |
| **Memória Limitada** | Analisa comportamentos passados gravados na memória. Toma decisões baseadas em experiência anterior. | Utilizada atualmente (ex.: carros autônomos). |
| **Teoria da Mente** | Identifica e compreende emoções, crenças, intenções e sentimentos humanos. | Em fase de desenvolvimento. |
| **Autoconsciente** | Desenvolve pensamentos e emoções próprias, sem pré-programação. Está ciente de seus próprios estados mentais. | Campo da hipótese (não realizado). |

## 5. Desafios da IA

| DESAFIO | DESCRIÇÃO |
| --- | --- |
| Processamento insuficiente de dados | Falta de dados de qualidade/quantidade suficientes. |
| Capacidade limitada de hardware | Poder computacional ainda é uma barreira. |
| Interface Homem-Máquina | Dificuldade na interação natural entre humanos e máquinas. |
| Custos altos | Desenvolvimento e infraestrutura são caros. |
| Considerações éticas | Vieses, privacidade, impacto social, etc. |

## 6. Áreas da IA

| ÁREA | DESCRIÇÃO |
| --- | --- |
| **Machine Learning (Aprendizado de Máquina)** | Algoritmos que aprendem com dados e melhoram com a experiência. |
| **Redes Neurais Artificiais** | Modelos inspirados no cérebro humano para processar informações. |
| **Processamento de Linguagem Natural (PLN)** | Interação entre máquinas e linguagem humana (tradução, reconhecimento de fala, análise de sentimento). |
| **Visão Computacional** | Algoritmos para entender e interpretar imagens e vídeos. |
| **Robótica Inteligente** | Combinação de IA e robótica para criar robôs interativos. |
| **Sistemas Especialistas** | Sistemas que imitam o conhecimento de especialistas humanos em domínios específicos. |
| **Algoritmos Genéticos** | Técnicas de otimização baseadas em evolução natural. |
| **Sistemas de Recomendação** | Algoritmos que sugerem itens com base em padrões de comportamento. |
| **IA Conversacional** | Chatbots e assistentes virtuais que interagem por diálogos naturais. |
| **Mineração de Dados** | Descoberta de padrões em grandes conjuntos de dados. |
| **Reconhecimento de Padrões** | Identificação de padrões (facial, objetos, imagens). |
| **IA Explicável (Explainable AI)** | Métodos que permitem entender e explicar decisões de sistemas de IA. |
| **Agentes Inteligentes** | Programas autônomos que tomam decisões em um ambiente específico. |
| **Aprendizado Profundo (Deep Learning)** | Subcampo do ML que usa redes neurais profundas para análise de dados. |
| **Processamento de Sinais** | Análise e interpretação de áudio, vídeo e outros sinais. |
| **IA Responsável (Responsible AI)** | Desenvolvimento e implementação ética e legalmente responsável da IA. |
| **Raciocínio Automatizado** | Algoritmos para automatizar o processo de raciocínio lógico (prova de teoremas). |

## 7. Questões de Concurso Comentadas

### 7.1 (QUADRIX/2022)

**Item:** "O conceito de inteligência artificial (IA) refere-se, unicamente, a duas grandes áreas do conhecimento: ciência da computação e matemática."

**Gabarito: Errado (E).**

**Comentário:** A IA é **multidisciplinar** – envolve também áreas como medicina, psicologia, neurociência, filosofia, etc.

### 7.2 (QUADRIX/2022)

**Item:** "Em uma visão ampla, a IA pode ser dividida em duas categorias principais: machine learning e deep learning."

**Gabarito: Certo (C).**

**Comentário:** Em uma visão ampla, IA = Machine Learning + Deep Learning (deep learning é subárea do ML).

### 7.3 (CESPE/CEBRASPE/2022)

**Item:** "As aplicações em inteligência artificial são definidas como uma subárea da área de aprendizagem de máquina (machine learning)."

**Gabarito: Errado (E).**

**Comentário:** **Machine Learning é uma subárea da IA**, e não o contrário. A IA é o campo mais amplo; o ML é uma de suas subáreas.

### 7.4 (QUADRIX/2023)

**Item:** "A inteligência artificial refere-se a um campo de conhecimento que não está associado à aprendizagem, uma vez que esta é uma capacidade puramente humana."

**Gabarito: Errado (E).**

**Comentário:** A IA **está associada à aprendizagem** – o aprendizado de máquina é um dos pilares da IA.

### 7.5 (FGV/2023 – Responsible AI e Explainable AI)

**Item:** "Responsible AI é sinônimo de IA Explicável."

**Gabarito: Errado (c) (a afirmativa incorreta é a c).**

**Comentário:**
- **IA Explicável (Explainable AI):** capacidade de explicar como uma decisão foi tomada.
- **IA Responsável (Responsible AI):** desenvolvimento ético e legalmente responsável, considerando impactos sociais.
- **Não são sinônimos:** uma IA pode ser responsável, mas ainda ser opaca (não explicável).

### 7.6 (FGV/2021 – Sinapses/TJRO)

**Item:** "Tecnologia que possui a capacidade de melhorar o desempenho por meio da experiência usando dados de treinamento, podendo ser supervisionado ou não, é o(a):"

**Gabarito: e) Aprendizado de Máquina (Machine Learning).**

**Comentário:** ML é a área da IA que aprende com dados de treinamento (supervisionado ou não).
