# Anotações – Grandes Modelos de Linguagem (LLM) e IA Generativa

## 1. IA Generativa – Conceito

- **IA Generativa:** sistemas de inteligência artificial projetados para **criar conteúdo** que é indistinguível do criado por humanos.
- **Alimentação:** grandes quantidades de dados – aprendem a replicar padrões complexos em diferentes tipos de mídia.
- **Produção:** novos conteúdos coerentes, relevantes e, em muitos casos, personalizados.

> ### 1.1 IA Generativa vs. IA Discriminativa
> - **IA Generativa:** concentra-se na **criação** de dados e conteúdo novo.
> - **IA Discriminativa:** concentra-se na **classificação e previsão** com base em dados existentes.

## 2. Tipos de Conteúdo Gerado por IA

| TIPO | EXEMPLOS |
| --- | --- |
| **Textos** | Poemas, histórias, roteiros, artigos, e-mails, cartas |
| **Imagens** | Fotos realistas, pinturas, desenhos, logos |
| **Vídeos** | Animações, montagens, efeitos especiais |
| **Áudios** | Músicas, sons ambiente, imitações de voz |
| **Código** | Scripts, programas, APIs |
| **Matemática** | Fórmulas, equações, gráficos |

| CATEGORIA | EXEMPLOS |

## 4. Problemas e Desafios da IA Generativa

| PROBLEMA | DESCRIÇÃO |
| --- | --- |
| **Viés algorítmico** | A máquina aprende com base em dados que podem conter erros ou preconceitos. |
| **Privacidade** | Uso de dados pessoais para treinar modelos. |
| **Transparência** | Dificuldade em saber de onde o conteúdo gerado foi baseado. |
| **Alucinações** | A IA ainda comete erros e gera informações falsas ou inconsistentes. |
| **Deep Fake** | Manipulação de mídia (imagens, vídeos, áudios) para criar conteúdo falso. |
| **Direitos autorais** | Questões sobre propriedade intelectual do conteúdo gerado. |
| **Perda de emprego** | Substituição de tarefas humanas por automação. |
| **Manipulação de eleições** | Uso de IA para gerar desinformação. |
| **Identidade falsa** | Criação de perfis e identidades fictícias. |

## 5. Ética na IA Generativa

| QUESTÃO ÉTICA | DESCRIÇÃO |
| --- | --- |
| **Deep Fake e manipulação de mídia** | Criação de imagens/vídeos falsos realistas. |
| **Violação de privacidade** | Uso indevido de dados pessoais. |
| **Viés e discriminação** | Perpetuação de preconceitos presentes nos dados. |
| **Direitos autorais** | Quem detém os direitos sobre conteúdo gerado por IA? |
| **Perda de emprego** | Automação de tarefas anteriormente humanas. |
| **Manipulação emocional** | Uso de IA para influenciar emoções e opiniões. |
| **Desigualdade digital** | Acesso desigual a tecnologias de IA. |
| **Ataques cibernéticos** | Uso de IA para gerar ataques mais sofisticados. |

> ### 5.1 Deepfake
> - Técnica que sintetiza imagens e sons por meio de IA.
> - Exemplo: recriação da imagem e voz de artistas falecidos para comerciais.

## 6. Síntese para Revisão Rápida

| CONCEITO | DEFINIÇÃO |
| --- | --- |
| **IA Generativa** | Criação de conteúdo novo (texto, imagem, áudio, vídeo). |
| **IA Discriminativa** | Classificação e previsão com base em dados existentes. |
| **LLM (Large Language Model)** | Modelos de linguagem de grande escala (ex.: GPT). |
| **ChatGPT** | Modelo de linguagem para geração de texto. |
| **Deepfake** | Criação de conteúdo falso realista com IA. |
| **Alucinação** | Geração de informações falsas pela IA. |
| **Viés Algorítmico** | Reflexo de preconceitos humanos nos dados de treinamento. |
