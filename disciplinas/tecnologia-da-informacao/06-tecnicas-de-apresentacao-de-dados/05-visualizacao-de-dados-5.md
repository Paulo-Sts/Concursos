# VISUALIZAÇÃO DE DADOS – FUNÇÕES E APLICAÇÕES

## 1. Funções da Visualização de Dados (Qual Gráfico Usar?)

A escolha do gráfico depende da **função** que ele precisa exercer sobre os dados.

| FUNÇÃO | DESCRIÇÃO | EXEMPLOS DE GRÁFICOS |
| :--- | :--- | :--- |
| **Fazer Comparações** | Mostrar diferenças ou semelhanças entre variáveis/valores. | Gráfico de Barras, Gráfico de Linhas, Gráfico de Radar. |
| **Mostrar Proporções** (Parte/Todo) | Usar tamanho ou área para mostrar partes em relação a um total. | Gráfico de Pizza, Gráfico de Rosca, Mapa de Árvore. |
| **Mostrar Conexões** | Mostrar relacionamentos, correlações ou interseções entre dados. | Diagrama de Venn, Gráfico de Dispersão. |
| **Mostrar Hierarquias** | Mostrar como os dados são classificados e ordenados. | Organograma, Mapa de Árvore (*Treemap*). |
| **Mostrar Conceitos** | Explicar e mostrar ideias ou conceitos. | Fluxograma, Mapa Mental. |
| **Mostrar Localizações** | Exibir dados sobre regiões geográficas. | Mapa Coroplético, Mapa de Bolhas, Mapa de Conexões. |
| **Mostrar Distribuição** | Exibir frequência e como os dados se espalham em um intervalo. | Histograma, Box-Plot, Gráfico de Densidade. |
| **Mostrar Movimentos/Fluxos** | Mostrar o movimento ou fluxo de dados entre estados. | Diagrama de Sankey, Mapa de Conexões. |
| **Revelar Padrões** | Revelar formas ou padrões nos dados. | Gráfico de Área, Gráfico de Barras, Mapa de Pontos. |
| **Mostrar Variação Temporal** | Mostrar a evolução de dados ao longo de um período. | Gráfico de Linhas, Gráfico de Velas (*Candlestick*), Gráfico de Gantt. |
| **Analisar Textos** | Revelar padrões e *insights* de corpos de texto. | Nuvem de Palavras (*Word Cloud*). |

## 2. Ferramentas, Princípios e Conceitos Cobrados em Prova

### 2.1 Ferramentas Específicas

- **Pesquisa e Visualização Web (CESPE/TRT-8/2022):** As ferramentas fornecidas, respectivamente, para mecanismos de pesquisa e visualização de dados de aplicações web são **Elasticsearch** (pesquisa) e **Kibana** (visualização).
- **Power BI (CESPE/DEPEN/2021):** O Power BI é uma ferramenta moderna utilizada para gerar *dashboards* de visualização de dados oriundos de fontes separadas.
- **Excel (OBJETIVA/Carlos Barbosa/2019):** O **Gráfico de Radar** é um tipo de gráfico disponível no Excel.

### 2.2 Princípios de Design e Visualização

- **Uso da Cor (CESGRANRIO/ELETRONUCLEAR/2022):** Em projetos de visualização, o uso da cor nos gráficos numéricos deve priorizar a **ênfase na informação a ser destacada** pelo gráfico.
- **Impacto Visual (CESPE/ANP/2022):** Um dos princípios fundamentais é o impacto visual, onde as informações com **maior relevância** devem ser **facilmente distinguidas** das de menor relevância.

### 2.3 Identificação de Gráficos Específicos

- **Box-Plot (CESPE/ANP/2022 e CESPE/ME/2020):** Consiste em uma representação gráfica de dados numéricos a partir de seus quartis, permitindo a efetiva detecção visual de *outliers*. É denominado diagrama de caixa, e não "caixa de barra".
- **Mapa Coroplético (FGV/SEFAZ-AM/2022):** A visualização que utiliza mapas com sombreamento ou tonalidades para exibir como um valor difere em proporção em uma localização geográfica é o **mapa coroplético**.
