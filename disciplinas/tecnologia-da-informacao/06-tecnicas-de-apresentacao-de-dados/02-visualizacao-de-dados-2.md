## 1. Gráfico de Linhas

### 1.1 Definição e Uso

- **Descrição:** Representa dados contínuos ao longo do tempo, com pontos de dados conectados por linhas.
- **Quando Usar:** Para visualizar **tendências** temporais (ex: crescimento de vendas mensais, variação da temperatura ao longo do dia).
- **Interpretação:**
    - A **inclinação** da linha mostra a direção da tendência (aumento, diminuição ou estabilidade).
    - Picos indicam valores máximos; vales indicam valores mínimos.

## 2. Gráfico de Dispersão (*Scatter Plot*)

### 2.1 Definição e Uso

- **Descrição:** Usa **pontos** para representar a relação entre **duas variáveis numéricas** quantitativas. Cada ponto é um par de valores (x, y).
- **Quando Usar:** É a ferramenta ideal para analisar a existência de **correlação entre duas variáveis**, como horas de estudo vs. nota, ou salário vs. crédito concedido.
- **Interpretação:**
    - **Correlação Positiva:** Pontos sobem para a direita (quanto maior x, maior y).
    - **Correlação Negativa:** Pontos descem para a direita (quanto maior x, menor y).
    - **Ausência de Correlação:** Pontos distribuídos aleatoriamente.
    - Quanto mais próximos os pontos estiverem de uma linha reta, mais **forte** é a correlação.

## 3. Histograma

### 3.1 Definição e Uso

- **Descrição:** Gráfico de barras que mostra a **distribuição de uma variável contínua**. As barras representam faixas de valores (intervalos/classes), e a altura da barra indica a **frequência** com que os dados caem naquele intervalo.
- **Quando Usar:** Para visualizar a forma da distribuição dos dados (ex: distribuição de idades em uma população, distribuição de notas de uma prova).
- **Interpretação:** O formato do histograma ajuda a identificar se a distribuição dos dados é normal (curva de sino), assimétrica, uniforme, ou se possui mais de um pico.

## 4. Análise de Questões de Concurso

### 4.1 Associação de Gráficos a Funções (FUNDEP/UFOP/2024)

- **Enunciado:** Associar gráficos a suas utilidades: comparar categorias vs. visualizar tendências.
- **Gabarito:** **D) 2, 1, 1.**
    - **Gráfico de Barras (2):** Útil para comparar várias categorias.
    - **Gráfico de Linhas (1):** Usado para visualizar tendências e padrões em dados contínuos ao longo do tempo.

### 4.2 Identificação da Ferramenta Correta (FGV/TJ-DFT/2022)

- **Enunciado:** Identificar a ferramenta para analisar a correlação entre tempo de atendimento e experiência do funcionário.
- **Gabarito:** **D) gráfico de dispersão.**
- **Análise:** Esta é a função do gráfico de dispersão. O Gráfico de Pareto (80/20), o Diagrama de Ishikawa (causa-efeito) e o Gráfico de Gantt (cronograma) não revelam correlação entre duas variáveis numéricas.

### 4.3 Gráfico de Pizza e Variável Qualitativa (CESPE/FUB/2022)

- **Enunciado:** O gráfico de pizza é o mais apropriado para representar dados de disciplinas por semestre, pois a variável é qualitativa ordinal.
- **Gabarito:** **ERRADO.**
- **Análise:** "Número de disciplinas" (2, 3, 4, 5...) é uma variável **quantitativa discreta**, não qualitativa ordinal. Para este tipo de dado, um **gráfico de colunas (barras)** ou um histograma seria mais apropriado.
