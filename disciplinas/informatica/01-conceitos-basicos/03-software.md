# Software

## 1. Conceito de Software
- É a parte lógica do computador: o conjunto de instruções e programas que comandam o funcionamento do hardware.
- Sequência de instruções escritas para serem interpretadas pelo computador para executar tarefas específicas.

## 2. Classificação dos Softwares
- Os softwares são divididos em categorias conforme sua finalidade:
  - Software Básico: Define o padrão de comportamento do equipamento e permite a operação do computador. O principal exemplo é o Sistema Operacional (Windows, Linux, macOS).
  - Software Utilitário: Programas que servem para a manutenção ou otimização do computador. Exemplos: Antivírus, compactadores de arquivos (WinZip/WinRAR) e desfragmentadores de disco.
  - Software Aplicativo: Programas com funções específicas para o usuário final. Exemplos: Editores de texto (Word), planilhas (Excel), navegadores (Chrome) e jogos.
  - Software de Programação: Ferramentas utilizadas por desenvolvedores para criar outros programas. Exemplos: Compiladores, editores de código e IDEs.
  - Software Embarcado (Firmware): Software gravado diretamente no hardware (chips) para controlar dispositivos específicos, como o sistema de um micro-ondas ou de uma impressora.

## 3. Tipos de Licenças e Distribuição

#### Software Livre (Free Software)
- Baseia-se em quatro liberdades: executar, estudar (acesso ao código-fonte), modificar e distribuir (incluindo versões modificadas).
- Pode ser pago ou gratuito, mas o código deve ser aberto.

#### Software de Código Aberto (Open Source)
- Similar ao software livre, mas com foco nos benefícios práticos e técnicos do desenvolvimento colaborativo.

#### Software Proprietário
- O uso, redistribuição ou modificação são proibidos ou restritos pelo dono. O código-fonte é fechado.

#### Formas de Distribuição Comuns:
- Freeware: Software gratuito para uso, mas o código-fonte permanece fechado e o autor retém os direitos.
- Shareware: Software disponibilizado para teste por tempo ou funcionalidades limitadas. Após o período, exige pagamento (ex: WinRAR).
- Adware: Software gratuito que exibe propagandas para se financiar.
- Trial: Versão de demonstração que para de funcionar após um período determinado.

## 4. Pontos-Chave para Concursos
- Software Livre vs. Gratuito: Software livre refere-se à liberdade (código aberto), não necessariamente ao preço. Já o Freeware é sobre preço (grátis), mas com código fechado.
- Sistema Operacional: É a base (Software Básico). Sem ele, o hardware não funciona para o usuário.
- Firmware: É o software "dentro" do hardware. Exemplos clássicos são a BIOS do computador.
- Questões de "Somente/Apenas": Cuidado com afirmações que dizem que "todo software livre é gratuito" ou "todo software gratuito é livre"; ambas costumam estar incorretas.
